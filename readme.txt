== Description ==

**BrandDigital Team Showcase** is the ultimate team block plugin for Gutenberg, designed to help you build beautiful, professional, and highly engaging "Meet the Team" sections, staff showcases, and company directories in minutes—completely code-free! 

Whether you are an agency, startup, corporate site, or freelancer, this plugin offers premium, pixel-perfect layouts and styling controls that fit seamlessly into any WordPress theme. Built with modern web design trends and performance in mind, it utilizes vanilla JavaScript (no jQuery) to ensure lightning-fast page loading and maximum SEO optimization.


### 🌟 Key Benefits:
* **Gutenberg Native:** Works directly within the block editor. No bulky or bloated third-party page builders needed.
* **100% Responsive:** Optimized for all devices—looks outstanding on mobile, tablet, and desktop screens.
* **Ultra-Lightweight & Fast:** Zero jQuery dependencies, clean CSS/JS, and built-in lazy loading for elite performance.
* **Unlimited Styling Options:** Customize every detail to match your brand identity with typography, margins, and borders.

### ✨ Free Features & Layouts

* ** 8 Elegant Pre-Designed Templates:**
  * *Corporate & Agency:* Ultra-clean, professional grids for modern business websites.
  * *Creative Hover & Image Overlay:* Sleek CSS transitions that reveal details elegantly.
  * *Minimal Card & Centered Profile:* Classic, high-end card styles with smooth alignments.
  * *Gradient Modern & Dark Mode:* Vibrant gradients and high-contrast dark theme presets.
  * *Split Layout & Compact Grid:* High-density layout options ideal for larger teams.
* ** Add Unlimited Team Members:** Showcase your entire team or organization with no arbitrary limits.
* ** Versatile Profile Pictures:** Style photos with Square, Rounded, or Circle layouts, complete with borders.
* ** Dynamic Social Media Profile Links:** Seamlessly link to Facebook, LinkedIn, Twitter/X, and custom websites.
* ** One-Click Direct Contact Buttons:** Drive quick actions with dedicated Email, Phone, and WhatsApp buttons.
* ** Gorgeous Hover Animations:** Interactive hover effects, subtle card zoom, and fade-in stagger page load options.
* ** Highlight "Featured Member" Badge:** Add customizable badges (e.g., "CEO", "Founder", "Manager") to specific cards.
* ** Intuitive Layout Customizer:** Control grid columns (1 to 4 columns), row gaps, margins, and content alignment.
* ** Premium Color Controllers:** Tweak colors for card backgrounds, text, names, designations, and hover elements.
* ** Flexible Visibility Toggles:** Toggle on/off display for bios, social links, contact info, and badges.
* ** Drag-and-Drop Reordering:** Easily rearrange staff members with up/down arrows directly inside the Gutenberg editor.
* ** Translation & RTL Ready:** Ready to localize into any language with native RTL text support.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wpbranddigital-team-showcase` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the block editor, search for "WPBrandDigital Team Showcase" and add it to your page.
4. Add team members, choose a layout, and customize to your liking.

== Frequently Asked Questions ==

= How many team members can I add? =

There is no limit. You can add as many team members as you need within a single block.

= Can I reorder team members? =

Yes! Use the up/down arrow buttons in the editor to drag and reorder members easily.

= Is it responsive? =

Absolutely. The block is fully responsive and looks great on all devices.

= Does it work with any theme? =

Yes, WBD Team Member is designed to work with any properly coded WordPress theme.
